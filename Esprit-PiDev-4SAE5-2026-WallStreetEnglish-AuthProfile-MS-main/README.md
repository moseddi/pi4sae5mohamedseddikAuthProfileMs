# WallStreetEnglish-Backend – Auth & Profile Microservices

## Overview
This project was developed as part of the **PIDEV – 4th Year Engineering Program** at **Esprit School of Engineering** (Academic Year 2025–2026).

It consists of a backend microservices architecture for the WallStreetEnglish language school platform, handling authentication and user profile management with two dedicated microservices.

## Tech Stack
- Spring Boot
- Spring Security + JWT
- MySQL
- Spring Data JPA
- Maven

## Microservices

**Auth Service** (Port: 8081)
- User registration & login
- JWT token generation
- Password encryption

**Profile Service** (Port: 8082)
- Profile management
- User information CRUD
- Profile picture upload
